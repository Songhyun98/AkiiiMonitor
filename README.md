# 아키모니터 · AKIIIMONITOR
**아키클래식(AKIII CLASSIC) 브랜드 건강검진 대시보드**

🔗 **[대시보드 바로가기](https://akiiimonitor-jkb3nnfumu75enda2zjdmc.streamlit.app/)**

---

## 프로젝트 개요

라이프스타일 스포츠 브랜드 **아키클래식**의 채용 공고 대응 미니 프로젝트입니다.
네이버 데이터랩 검색 트렌드 및 블로그·뉴스·카페 언급 데이터를 기반으로,
브랜드의 검색 점유율·성장성·시장 포지셔닝을 자동으로 진단하는 인터랙티브 대시보드를 구현했습니다.

- **분석 기간**: 2023년 1월 ~ 2026년 5월
- **데이터 갱신**: 주간 자동갱신 (완료월 컷오프 자동 적용)
- **제작**: 유수연

## 주요 분석 결과

| 지표 | 수치 |
|---|---|
| 장기 성장률 (6M 블록) | **+18%** |
| 최근 모멘텀 (3M 평활 YoY) | **+3.4%** |
| 니치 마켓 검색 점유율 (SoS) | **85.4%** |
| 소비자 견인력 GAP (SoS−SoV) | **+36%p** |

### 핵심 인사이트
- **니치 마켓 독점**: 동급 DTC 마켓(포즈간츠·23.65) 내 검색 점유율 85.4% — 독점적 지배력 확보
- **성장 엔진**: 컴포트·여행 수요 침체(−46%, −49%) 속, 아키클래식이 베팅한 발 건강 수요만 유일하게 성장(+26%)
- **체급 진입**: 스케쳐스·휠라 등 메이저 브랜드가 정체인 반면, 아키클래식은 +18% 구조적 성장
- **전술 시사점**: 11월 검색 수요 피크(지수 129) 프론트로딩 + 카페 커뮤니티(29%) 비수기 방어

## 대시보드 구성

| 섹션 | 내용 |
|---|---|
| **① 니치 마켓 독점** | 검색 점유율(SoS) 추이 · 성장 지수 · SoS−SoV GAP |

## 분석 방법론

- **SoS (Share of Search)**: 동급 니치 마켓 내 검색량 비중 — 절대 크기(점유) 지표
- **성장 지수 (재인덱싱)**: 자기기간 평균=100 정규화 — 추세 방향 지표 (크기 비교 X)
- **장기 성장률**: 최초 6개월 대비 최근 6개월 평균 검색량 변화율
- **3M 평활 YoY**: 단일월 노이즈·기저효과 보정 — 모멘텀 헤드라인 지표
- **SoS−SoV GAP**: 검색 점유 대비 언급 점유 차이 — 능동적 수요 vs 버즈 효율 진단
- **완료월 컷오프 (ASOF)**: 진행 중인 달 자동 제외 → 미완성 데이터 착시 방지
